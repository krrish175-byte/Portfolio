# Krrish Biswas Portfolio

A high-performance, dark-themed personal portfolio built with React, Vite, Tailwind CSS, and Framer Motion.

## Features

- **Terminal Hero**: Animated typing effect showing off open-source achievements.
- **Scroll Animations**: Smooth fade-ins and scale effects using Framer Motion.
- **Live Stats**: Animated counters for PRs, orgs, and mentorships.
- **Mentorship Focus**: Detailed cards for LFX (OpenSSF) and GSoC (Linux Foundation).
- **Contribution Graph**: Mock GitHub-style heatmap for visual impact.
- **Project Showcase**: Detailed case studies for key open-source projects.
- **Responsive Design**: Fully optimized for mobile, tablet, and desktop.
- **Performance**: Near-perfect Lighthouse scores with minimal dependencies.

## Tech Stack

- **Framework**: React 18
- **Build Tool**: Vite
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Icons**: Lucide React

## Getting Started

1. **Install dependencies**:
   ```bash
   npm install
   ```

2. **Run development server**:
   ```bash
   npm run dev
   ```

3. **Build for production**:
   ```bash
   npm run build
   ```
